Projeto desenvolvido em C que utiliza grafos para representar uma rede de antenas numa cidade. Cada antena é um vértice ligado a outros com a mesma frequência. Permite leitura de dados a partir de ficheiro e execução de operações como buscas em profundidade/largura, caminhos entre antenas e interseções entre frequências distintas.

Detailed Description

Responsável pela lógica específica para determinados objetivos do sistema.

Este ficheiro contém funções lógicas que:

  • Realizam travessias em profundidade (DFS) e em largura (BFS);
  • Verificam todos os segmentos existentes (apenas válidos entre dois vértices da mesma frequência);
  • Determinam todas as interseções de arestas entre frequências diferentes.

Realiza uma travessia em largura (BFS) num subgrafo com determinada frequência, a partir de um id de vertice.

Função de travessia em largura (BFS).

A função percorre o subgrafo correspondente à frequência indicada, a partir de um vértice inicialpré-selecionado pelo user, imprimindo todos os vértices adjacentes em cada nível antes de avançar para os seguintes. Utiliza uma fila (FIFO) para gerir os vértices por visitar.

Returns
