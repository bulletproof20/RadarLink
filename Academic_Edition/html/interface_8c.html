Projeto desenvolvido em C que utiliza grafos para representar uma rede de antenas numa cidade. Cada antena é um vértice ligado a outros com a mesma frequência. Permite leitura de dados a partir de ficheiro e execução de operações como buscas em profundidade/largura, caminhos entre antenas e interseções entre frequências distintas.

◆ MostrarIntersecoes()

void MostrarIntersecoes ( LL_Intercessoes inicioI,
LL_segmento inicioSg,
LL_subgrafos inicioS )

Mostra as interseções entre segmentos de diferentes frequências.

Para cada interseção existente, são identificados os dois segmentos envolvidos. A função imprime os vértices associados a cada segmento, bem como as respetivas coordenadas, assinalando visualmente a interseção.

Em caso de erro na procura de segmentos ou vértices, termina a execução através de FalhaMalloc().

Parameters
[in]inicioIPonteiro para o início da lista de interseções.
[in]inicioSgPonteiro para o início da lista de segmentos.
[in]inicioSPonteiro para o início da lista de subgrafos.

< Ponteiro auxiliar para percorrer a lista de interseções

< Itera sobre cada interseção da lista

< Itera duas vezes: uma para cada segmento da interseção

< Avança para a próxima interseção
