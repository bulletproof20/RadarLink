Detailed Description

Definição de estruturas de dados para manipulação de antenas, arestas e outros componentes gráficos.

Este ficheiro contém as definições das estruturas de dados utilizadas no processamento e manipulação das antenas, arestas, vértices, segmentos e interseções, de forma eficiente.
