Projeto desenvolvido em C que utiliza grafos para representar uma rede de antenas numa cidade. Cada antena é um vértice ligado a outros com a mesma frequência. Permite leitura de dados a partir de ficheiro e execução de operações como buscas em profundidade/largura, caminhos entre antenas e interseções entre frequências distintas.
