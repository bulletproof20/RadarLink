# Academic Context

RadarLink was developed as a practical assignment in the unit **Estruturas Avançadas de Dados (EAD)** — *Advanced Data Structures* — within the **Engenharia de Informática Médica (EIM)** programme at **IPCA** (*Instituto Politécnico do Cávado e do Ave*).

| Detail | Value |
|--------|-------|
| **Unit** | Estruturas Avançadas de Dados (EAD) |
| **Programme** | Engenharia de Informática Médica (EIM) |
| **Institution** | IPCA |
| **Academic year** | 1st year |
| **Semester** | 2nd semester |

## First contact with C

This project was one of the first **independent, hands-on experiences with the C language** during the degree. The work required managing memory manually, designing dynamic data structures, and reasoning about algorithms without the safety nets of higher-level languages.

## Beyond the assignment brief

The development did not follow only the formal requirements of the *enunciado*. It also reflected personal curiosity — experimenting with graph models, geometric intersection logic, and file-driven city layouts — and a willingness to explore concepts that went slightly beyond the minimum scope. That combination of curricular obligation and technical interest shaped how the project grew from checkpoint to deliverable.
